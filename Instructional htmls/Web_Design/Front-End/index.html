Hello Beginner! Welcome!

Hello! Welcome!

This page contains a roadmap of how to develop the fornt-end interfaces for web apps.

For front-end, you almost certainly need to be very well trained in HTML, CSS, and JavaScript. This is because; everything else connected to front-end engineering is connected to these, and your knowledge of these will enable you to create relevant components for your interfaces.

As you move on, you might need to learn low-code tools like WordPress and Shoppify. A knowledge in these might give you an easier entry in the market as a freelancer. But as discussed above, a working knowledge of HTML, CSS, and JavaScript is crucial in order to make you conversant with various components of webpages.

You'll learn HTML in order to be able to dissect and add to a webpage. A foundational knowledge is enough, you'll learn more as you progress with other technologies. You just need to know its basic structure and basic tags and you're good to go.

You'll learn CSS in order to learn to beautify your webpages, and it is here where you'll learn various components of a webpage and how they're used. Knowing design elements (eg. a header, hero, navigation bars, etc.) will assist you very much in knowing how to style a webpage to look exactly how you want it to look. A knowledge of CSS basics is crucial here, as all the styling will depend on you knowing the relevant parameters. But that's just about it. Once you know which parameters to style, you are ready to learn one of the CSS frameworks out there.

Frameworks (CSS frameworks in particular) are important to you because you really don't know any JavaScript up to now. Most of the design elements you'll want to use on your page require some JavaScript to work. CSS frameworks come with their JavaScript bundled with them, so you don't have to write your own JavaScript code. So why learn JavaScript?

As you move up on the ladder of front-end development, your focus usually shifts from making a webpage into making components in a webpage. That's when you would want to know JavaScript.
