Font

  • You can style the typeface by using the following properties:
    • font-family: for the type of typeface you want.
    • font-weight: for a light, regular, or bold font. Keywords are: bold, normal, lighter, bolder
    • font-weight can also use numbers to indicate the font weight desired, 1 being the lightest and 1000 the boldest. People usually use increments of 100. Look if the font you are using accepts numeric values though; Not all of them accept numerical values and not all numerical values can be expressed to all fonts.
    • font-size: for the size. Make sure to add the "px" at the end of the font size you just entered. Eg p {font-size: 18px;}
    • With font-style, you can italicize an entire block of text. The default style is normal. The fancier one is italic
    • font-decoration: allows you to underline text. Check here for a list of rules you can set for this property.
    • text-transform changes text to all uppercase or lowercase. Eg. p { text-transform: uppercase; }
    • text-align: for text alignment. It accepts left, center, right, and justify as arguments. Eg.
    • letter-spacing puts space between letters. The value is give in pixels. Eg:
    • word-spacing puts spacing btn words, as it says. The value is also given in pixels. Eg:
    • line-height controls how tall are our lines. A line of text is comprised of 1). The font size, and 2). the leading. line-height controls the height of the combination of those two, by mainly increasing the height of the leading space. The value can either be given in unitless values, pixels, or ems, but unitless values are preferred as they are responsive to the change in font size (changing the font-size will automally change the line height)
    • color: This changes the color of the text itself. background-color changes the color of the text box behind the text.
    • opacity: This controls the opacity of the text-box. Use this if you want to create a beautiful overlay effect
    • background-image is able to change the background of the text-box to be a picture.

Linking Fonts From The Web

  • There are two ways this can be done: Linking through the html document, or linking through the CSS document.
Linking in the index HTML

  • This can be done by using the link tag; you just need to place it inside the head tag and then set the href attribute equal to the link where your fonts are

  • There are free fonts at Google Fonts and Adobe Fonts. Take a look.
Linking Through the CSS file
  • This can be achived by using the @font-face {} ruleset. It takes all the normal font formatting properties, plus the src: url() format() that specifies where your font is, and which font format will be used.
  • There are 4 main font formats: OpenType Font (OPT), TrueType Font (TTF), Web Open Font Format (WOFF), and Web Open Font Format 2 (WOFF2). Check this site for an in-depth understanding of font formats.
  • Since the best font format for web (because it loads faster) is WOFF2, but free font providers such as Google Fonts and Adobe Fonts will likely give you a .ttf font format, it becomes imperative for you to download the font families you want, generate the additional formats via a free font generator like this one, and then download the font formats file, unzip it, and save the contents in the same directory as your web files.
