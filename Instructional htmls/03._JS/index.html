    • Logical Operators

  • They are:
    • The && operator: Checks if each of the two arguments evaluated to true. When it does, the statement itself will evaluate to true
    • The || operator: Checks if one of the two statements given evalueates to true. If it does, the whole statement evalueates to true and the subsequent code is run.
    • The ! operator: negates the condition put inside the brackets. If the condition is false, the ! bang operator will bring it as true and vice versa.
