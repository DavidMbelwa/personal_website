Order of Execution

  • CSS prioritize ID's over class or element type styling.
  • That is to say, if you had styled your paragraph in a section of HTML to be red, and then styled font color of the section to be black by using a section ID, your paragraph will be black.
  • So when styling, start by targeting the element type first. This will apply the styling to all similar elements of course, but you can go after the ones you specifically want by assigning classes to them or assigning ID's
