Before You Start:

  • Always design for mobile first.
  • In the same spirit, set your columns first (their width) and then specify how many columns each element will span in the HTML. Do this for every major section of your document. If you've set the width of columns, but has forgotten to assign a major section a number of columns it will span, that section will start at the top, acting as an inline, and in some cases engulf other elements inside it. You don't want that.

What is CSS?

  • You use CSS to format your web page. This ranges from things like setting color and font-families to to your elements to things like setting the width, size, and position of elements in your document.
