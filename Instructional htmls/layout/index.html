Links and Buttons
  • Don't forget to add tooltips to your links, so that a user can get a gist of what a link will be about.
  • Style your links using color and the text-decoration property so that they stand-out but don't diverge very far from the overall color theme of the site.
  • Add hover effects using the :hover pseudo-class of CSS on your element.
  • Add changes to the cursor using the cursor: ; property.
