The Flex Box

  • It is a styling system that makes it possible for HTML to resize based on the size of the screen.
  • It is set by setting the display property of an element to either "flex" or "inline-flex". This will make the element a flex container
  • The flex container has properties that enable items within it to resize and re-position themselves based on how stretched the box is.
  • display: flex; will make the container a block element while the display: inline-flex; will make it an inline-block.
  • Once that is done; you can go inside and start commanding how the elements within the container should behave. The first property is justify-content.
  • justify-content can take any of the following values:
    • flex-start: Which will align everything in the flex container to the left.
    • flex-end: which will align all items in the flex container to the right.
    • center: As is self-explanatory, will center the content of a flex container
    • space-around: which will put equal space on the sides (left-right) of each element, with regard to margins also.
    • space-between: which will put equal space between items themselves, but no space at the right side of the left-most item and the left-side of the right-most item (the last item)
  • With justify-content, you're basically instructing the browser to stuck together items in an inline fashion in the event the screen is squeezed out. Read further down to learn how you can stretch the items vertically or horizontally.
  • align-itmes: is another property applied to the container. It aligns the items vertically and takes the following rulesets:
    • flex-start: which aligns the items on the top of the container.
    • flex-end: which aligns the items on the bottom.
    • center: centers, as it says.
    • baseline: which aligns the bottoms of each element
    • stretch: which stretches the items the whole height of the container (will not go beyond the max-height instructions though.).
  • The flex-grow property allows items to expand and fill the unused space in the container in both the vertical and horizontal direction. It is set in numbers, and when you set display: flex; it is automatically set to 0.
  • It is also used to regulate the rate of how much an item expands in relation to other items in the same container, so if there were two items in the flex container, and one was assigned the value of 1 and the other 2, the second will expand twice as much as the first one. And not to forget, flex-grow is set on a flex item, not container.
