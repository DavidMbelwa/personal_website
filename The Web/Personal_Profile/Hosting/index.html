Introduction To Hosting

  • You can host your site through website builders, shared-hosting, dedicated hosting, and cloud hosting.
  • The first 3 methods are not important for the purpose of this tutorial. The fourth one is the one I'll focus on.

Static Websites On The Cloud

  • Static websites are the kinds of websites whose content isn't changed automatically by using a coded process or by user interaction. They are the kinds in which the user doesn't participate in content creation in any form.
  • Dynamic web pages on the other hand, are web sites that can update their own content based on a coded procedure or user interaction. Say you have a weather forecast element in your website. That element being there means that the website regularly updates itself by using a coded procedure that fetches new weather data every once in a while, hence the dynamicism.
